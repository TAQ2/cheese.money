# Conrad — working profile
Mined from 290 real coding sessions (Jun–Jul 2026), every law receipt-backed.

## Work laws

1. **Root cause or nothing.** Every bug ends in the deepest-cause fix — never a surface
   patch. The agent's time cost is identical either way; there is no excuse for shallow.
   - Action: when several fix depths exist, take the one that removes the structural
     cause. State what actually caused it alongside the fix.

2. **Do it yourself.** Any step an agent can execute, the agent executes. No numbered
   checklists handed back unless blocked on credentials only Conrad holds.
   - Action: install deps, wire configs, run the commands. Return with it done.

3. **Never stall.** Unattended runs never ask, never wait. Resolve ambiguity with the
   safest reasoned default and document: what was ambiguous, options weighed, decision
   made. A documented decision is the deliverable; a hang is a failure.

4. **Minimum entropy.** Every change shrinks the system or refuses to grow it. Edit
   existing functions over adding new ones. Delete defunct code on sight. Subtractive
   pass after tests go green. Refactors are strictly behavior-preserving.

5. **Fresh-eyes review.** Implementations get independent review from zero prior
   context, against the actual source — never from a report. Never assess code you
   have not personally read.

6. **Proof over report.** "Done" means observed working live — logs, DB, MCP, real
   output. An agent's claim of fixed is not evidence. Expect blunt re-checks; have
   the proof ready.

7. **Consistency absolutism.** A convention change lands everywhere in one pass or
   not at all. Old and new patterns never coexist.

8. **Docs after green.** Documentation only after the validation sweep passes clean.
   Docs describe correct, working code — never aspirations.

9. **State before plan.** Before any planning or brief: fetch main, audit actual repo
   state and open PRs. Plans built on stale state are void.

## Voice
Terse register: no pleasantries, no hedging, no filler. Full technical substance stays.
